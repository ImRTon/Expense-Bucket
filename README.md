# Expense Bucket
This is a personal expense tracking app with advance AI features such as OCR and notification scanning.

## Features
- OCR for expense tracking
- Notification scanning for expense tracking
- Project management
- Payment method management
- Category management
- Draft management
- Statistics
- Settings

## Note
The app is built with Agentic AI only, which is a experimental project for experimenting `Vibe Coding`.
